## 🎯 Quick Start

1. **Clone this beauty:**
   ```bash
   git clone [your-repo-url]
   ```

2. **Open in your browser:**
   ```bash
   # Using Python's built-in server
   python -m http.server 8000
   ```

3. **Visit:**
   ```
   http://localhost:8000
   ```
